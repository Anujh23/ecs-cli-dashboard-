# ECS & CLI Performance Dashboard

A real-time performance tracking dashboard for ECS and CLI products with editable targets, animated progress bars, and a glassmorphism UI.

## Live Demo

[https://anujh23.github.io/ecs-cli-dashboard-/](https://anujh23.github.io/ecs-cli-dashboard-/)

## Features

- **Editable Targets** — Click the edit icon to update disbursement, collection, and combined targets inline
- **Animated Progress** — Smooth number animations and progress bar transitions on every update
- **4 Summary Cards** — Fresh Leads, Total Amount, Total Disbursed, Total Collected with product-wise breakdown
- **Product Comparison** — ECS vs CLI with leading/trailing status indicators
- **Dark Mode** — Toggle between light and dark themes (auto-detects system preference)
- **Persistent Data** — All values and targets are saved to localStorage
- **Responsive** — Works on desktop, tablet, and mobile
- **Quick Fill** — Sample data buttons for fast testing
- **Keyboard Shortcuts** — `Ctrl+Enter` to update dashboard

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (No dependencies)

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/Anujh23/ecs-cli-dashboard-.git
   ```
2. Open `index.html` in your browser

No build tools or dependencies required.
